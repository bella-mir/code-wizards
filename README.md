# Code-Wizards Team
CodenRock Hakathon 17.09.2022-18.09.2022


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/bella-mir/code-wizards.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the application in a development mode
   ```sh
   npm start
   ```

## Stack: 

* JavaScript 
* React.js
